# Usmanova Fit

Адаптивный лендинг из трёх экранов: hero, программы, гарантии + форма.

Стек: HTML, CSS, JS. Без фреймворков и сборщиков.

## Экраны

1. Hero — оффер, фото, CTA.
2. Тренировки дома — карточки программ с кнопками.
3. Гарантии + форма подбора — валидация, модальные окна, состояние успешной отправки.

## Запуск

Открыть `index.html` в браузере.

## Деплой на GitHub Pages

1. Загрузить файлы в корень ветки `main`.
2. Settings → Pages → Deploy from a branch.
3. Branch: `main`, folder: `/(root)`.
