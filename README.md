# Usmanova Fit — vibe coding demo

Демо для тестового задания.

Повторены 3 экрана лендинга:
1. Первый hero-экран.
2. Экран с карточками программ.
3. CTA-блок с формой подбора программы.

Что работает:
- адаптивная вёрстка под телефон;
- кнопки скроллят или открывают модальное окно;
- форма валидируется;
- данные сохраняются в localStorage;
- при деплое на Netlify форма может отправляться как Netlify Form.

Деплой:
1. Распаковать ZIP.
2. Загрузить папку `usmanova-vibe-test` в Netlify Drop.
3. Скопировать URL вида `*.netlify.app`.
