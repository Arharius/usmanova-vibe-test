# Usmanova Fit

Лендинг из трёх экранов: hero, программы, форма подбора.

Стек: HTML, CSS, JS. Без фреймворков и сборщиков.

## Экраны

1. Hero — оффер, фото, CTA.
2. Программы — карточки программ с кнопками.
3. Форма подбора — гарантии, поля заявки, валидация, состояние успешной отправки.

## Запуск

Открыть `index.html` в браузере.

## Деплой на GitHub Pages

1. Загрузить файлы в корень ветки `main`.
2. Settings → Pages → Deploy from a branch.
3. Branch: `main`, folder: `/root`.
