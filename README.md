# Usmanova Fit

Лендинг с тремя экранами: hero, витрина программ, форма подбора.

Стек: HTML, CSS, JS — без фреймворков и сборщиков.

## Экраны

1. **Hero** — заголовок, преимущества, CTA-кнопки, фото с плашкой статистики.
2. **Программы** — карточки с фото, бейджем и кнопкой «Подробнее» (открывает модальное окно).
3. **Форма подбора** — блок с гарантиями + форма с валидацией и состоянием успешной отправки.

## Форма

Работает на стороне клиента: валидация полей, сброс после отправки, отображение сообщения об успехе.

## Запуск

Открыть `index.html` в браузере — дополнительных зависимостей нет.

## Деплой на GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить файлы в корень ветки `main`.
3. В настройках репозитория → Pages → Source: `main / (root)`.
4. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.
